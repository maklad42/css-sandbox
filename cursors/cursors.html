General

auto
The UA will determine the cursor to display based on the current context.
default
The platform dependent default cursor. Typically an arrow.
none
No cursor is rendered.

Info

context-menu
A context menu is available.
help
Help information is available.

Action

pointer
The cursor is a pointer that indiciates a link. Typically an image of a pointing hand.
progress
The program is busy in the background, but the user cann still interact with the interface (in contrast to wait).
wait
The program is busy, and the user can't interact with the interface (in contrast to progress).

Select

cell
The table cell or set of cells can be selected.
crosshair
Cross cursor, often used to indicate selection in a bitmap.
text
The text can be selected. Typically the shape of a I-beam.
vertical-text
The vertical text can be selected. Typically the shape of a sideways I-beam.
alias
An alias or shortcut is to be created.
copy
Something is to be copied.

Movement

move
Something is to be moved.
no-drop
An item may not be dropped a tthe current location. On mac OS X this is the same as not-allowed.
not-allowed
The requested action will not be carried out.
grab
Something can be grabbed (dragged to be moved).
grabbing
Something is being grabbed (dragged to be moved).
all-scroll
Something can be scrolled in any direction (panned). On Windows, all-scroll is the same as move.

Resize

col-resize
The item/column can be resized horizontally. Often rendered as arrows pointing left and right with a vertical bar separating them.
row-resize
The item/row can be resized vertically. Often rendered as arrows pointing up and down with a horizontal bar separating them.
n-resize
The top edge is to be moved.
e-resize
The right edge is to be moved.
s-resize
The bottom edge is to be moved.
w-resize
The left edge is to be moved.
ne-resize
The top right corner is to be moved.
se-resize
The bottom right corner is to be moved.
sw-resize
The bottom left corner is to be moved.
nw-resize
The top left corner is to be moved.
ew-resize
Bi-directional resize cursor (left-right).
ns-resize
Bi-directional resize cursor (up-down).
nesw-resize
Bi-directional resize cursor.
nwse-resize
Bi-directional resize cursor.

Zoom

zoom-in
Something can be zoomed in on.
zoom-out
Something can be zoomed out from.
